# SweetCrumb Cakes Demo

SweetCrumb Cakes is a standalone fictional demo website created for Tristan Web Studio. It presents a polished local website concept for a Brampton-based home bakery that sells custom cakes, cupcakes, cookies, dessert boxes, and dessert tables.

## Business concept

SweetCrumb Cakes is imagined as a home bakery serving birthdays, baby showers, weddings, corporate events, and small parties. The site is designed to feel like a real small business presence while using only styled placeholder visuals, not real bakery photography or copyrighted images.

## Conversion strategy

The demo shows how a home baker can move visitors from scattered Instagram DMs and word-of-mouth messages into a clearer quote request flow.

- The hero and CTAs guide visitors toward requesting a quote.
- The gallery builds desire with visual examples.
- The flavours and starting prices reduce repetitive questions.
- The ordering steps explain the process before someone reaches out.
- The FAQ removes common hesitation around notice, delivery, deposits, and servings.
- The front-end-only order form collects useful quote details in one place.

## How to run locally

Open this file in a browser:

```text
sweetcrumb-cakes-demo/index.html
```

No build step, framework, backend, npm install, database, or form service is required.
