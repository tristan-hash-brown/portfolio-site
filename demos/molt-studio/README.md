# MOLT Studio Demo

MOLT Studio is a standalone fictional handmade jewelry demo for the Tristan Web Studio portfolio. It is built as a static HTML/CSS/JS site under `demos/molt-studio/`.

## Portfolio Copy

Demo description:

> MOLT Studio is a fictional handmade jewelry demo showing how a small maker can turn Instagram interest and custom requests into a bold branded website with product drops, custom inquiry flow, and editorial storytelling.

Demo strategy angle:

> From Instagram DMs to a bold product-drop experience.

Suggested portfolio category:

> Branding / Product Demo

Suggested thumbnail direction:

> Dark editorial jewelry flat-lay with sculptural rings, chain details, molten bronze accents, off-white typography, and a bold product-drop feel.

## Demo Behavior

Product and custom CTAs use the static contact section as the inquiry path. The contact form is demo-safe: JavaScript intercepts submission and shows a fictional success message without sending data to an external service.
