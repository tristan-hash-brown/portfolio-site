# Cutline Barber Co. Demo

Standalone fictional demo website for a modern neighbourhood barbershop serving a Mississauga/Brampton-style suburban audience.

## Portfolio Copy

Category:
`Local Service / Booking Demo`

Demo description:
`Fictional neighbourhood barbershop demo showing how clear pricing, barber profiles, reviews, hours, and mobile booking CTAs can turn local search traffic into calls and appointments.`

Case study angle:
`From "Do they take walk-ins?" to a clear booking path.`

## Demo Notes

- Fictional business, address, phone number, barbers, reviews, and performance claims.
- Built as a mobile-first one-page static demo.
- Primary conversion goal: book an appointment.
- Secondary actions: call the shop, view pricing, choose a barber, and get directions.
- No external dependencies or real booking endpoint are used.
- The booking request form is demo-safe, prevents normal submission with JavaScript, and does not send data externally.
